PDF Extraction

Using the coordinates of glyphs (or characters imprinted on a pdf file), I devised a java program to extract structured data from pdf files. This software worked better than the available market softwares, since it learned the lines mapped inside the pdfs to confidently detect the placement of each cell inside the tables. Just by providing the url link to the pdf and the header of your structured data, it extracts all the table in less than a second!

UI Functionality For Big Data

Using AngularJS, I developed a web application (for investors) which helps them to search through all the commodities available throughout the world (Equities, Exchange-Traded funds, Indexes etc.) and develop an optimum portfolio for their clients (depending on the risk each customer can withstand) using various models (eg Markowitz Model). Functionalities such as searching through thousands of commodities and adjusting different allocations on the screen itself along with rendering charts on the screen for big data were included in the application's user interface.

Machine Learning and Social Media Analytics

Feeding updated data from social media of various clients to my various models and then by machine learning, I was able to know when the client was travelling or moving to new place, having an occasion in their family, how the client was feeling or having plans to by a new commodity(car, house etc). By using this software, companies will be able to cross-sell and up-sell their products or insurances.
